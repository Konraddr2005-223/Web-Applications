Introduction to Web Applications

📌 Overview

This repository contains projects and assignments completed as part of the Introduction to Web Applications course. The goal of this repository is to demonstrate skills in building modern, responsive Full-Stack web applications.

🚀 Tech Stack

Below is a general list of technologies used in the projects within this repository (remove/keep the ones currently used in your classes):

Frontend (Client):

HTML5, CSS3, JavaScript (ES6+) / TypeScript

React / Angular / Vue.js

Tailwind CSS / Bootstrap

Backend (Server):

Node.js & Express / Java & Spring Boot / Python & FastAPI

REST API

Databases and Tools:

PostgreSQL / MySQL / MongoDB

Git & GitHub

Docker (for containerization)

Postman (for API testing)
